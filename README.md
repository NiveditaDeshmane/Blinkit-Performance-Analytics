# Blinkit Performance Analytics (Python EDA)

This project performs **Exploratory Data Analysis (EDA)** on Blinkitâ€™s retail dataset using Python to understand sales patterns, outlet performance, and product trends. The analysis helps uncover actionable insights that support data-driven business decisions and strategy optimization.

## Objectives
- Analyze overall sales performance
- Identify top-performing products and outlets
- Understand the impact of outlet size and location on sales
- Explore customer rating patterns

## Dataset Overview
The dataset contains information related to:
- Product details
- Outlet types, sizes, and locations
- Sales values
- Customer ratings

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analysis Performed
- Data cleaning and handling missing values
- Descriptive statistics
- Sales distribution analysis
- Outlet-wise and product-wise performance analysis
- Visualization of trends and patterns

## Key Insights
- Certain outlet types and sizes contribute significantly to total sales
- Product categories show varied sales performance
- Location-based trends influence revenue and customer ratings
